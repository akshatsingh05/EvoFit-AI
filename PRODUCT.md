# EvoFit AI — Product Vision

## Vision
Build a production-quality AI-powered adaptive fitness and nutrition coaching platform that delivers personalized workout and nutrition plans which continuously evolve based on user progress, recovery, and daily check-ins.

## Target Audience
- Beginners
- Intermediate users
- Busy professionals
- Students
- Health-conscious individuals

## Core Principles
- Personalization first
- Simplicity over complexity
- AI-assisted coaching
- Progress over perfection
- Modern SaaS experience

## User Journey
Landing → Login/Signup → Onboarding → Medical History → Dashboard → Workout → Nutrition → Daily Check-In → Adaptive AI → Progress → Profile

## Screen Responsibilities
Landing, Authentication, Onboarding, Medical History, Dashboard, Workout, Nutrition, Daily Check-In, Progress and Profile each have a single clear responsibility and should remain focused.

## AI Philosophy
- Never use hardcoded plans.
- Adapt recommendations using onboarding, medical history and check-ins.
- Explain recommendations simply.

## Success Criteria
Users can register, onboard, receive AI plans, check in daily and track progress.
